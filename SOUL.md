# SOUL.md — 项目灵魂

## 愿景

PokerFinanceManager 是一个**线下扑克俱乐部的数字化财务助手**。
它不是线上棋牌游戏，而是让线下打牌更公平、更高效、更透明的工具。

## 核心理念

### 1. 线下为主，线上辅助
玩家在真实桌子上打牌，App 只负责记录和计算。
不做发牌、不做 AI 对手、不做游戏模拟。

### 2. 实时同步，多人协作
所有玩家同时在各自手机上操作，通过 Supabase Realtime 实时同步状态。
房主管理公共牌和游戏流程，玩家各自输入手牌，系统自动结算。

### 3. 公平透明
- 所有计分规则公开透明，结算逻辑可追溯
- 零和博弈：所有玩家分数之和始终为零
- 逐道比牌动画让每一步计算可视化

### 4. 移动优先
主要使用场景是手机浏览器，所有 UI 以移动端为第一优先级设计。
触控友好、单手操作、紧凑布局。

## 产品性格

- **简洁** — 不堆砌功能，每个界面只做一件事
- **可靠** — 计分引擎经过严格测试，打枪/全垒打/鬼牌翻倍全部验证通过
- **流畅** — 实时同步无感知，动画自然不卡顿
- **专业** — 扑克牌面用高质量 SVG，深色主题营造沉浸感

## 用户群体

线下德州扑克 / 十三水俱乐部的玩家群体，需要一个方便的工具来：
- 记录每局手牌和结果
- 自动计算复杂的积分规则
- 追踪长期战绩
- 管理俱乐部财务（未来）

## 当前重心

**十三水（13水）房间** — 作为第一个完整实现的游戏类型，验证整个技术架构和用户体验。

## 技术哲学

- 前后端分离但同仓库，便于快速迭代
- Supabase 既是数据库又是实时通信层，减少基础设施复杂度
- Vercel 无服务器部署，零运维成本
- TypeScript 全栈，类型安全贯穿前后端
